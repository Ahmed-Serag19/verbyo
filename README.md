Verbyo App: A Social Media Platform with Rewards and Impact
Verbyo App is a React-based web application designed to enhance your social media experience. It allows you to create and share content, get rewarded for your contributions, and amplify social causes you care about.  The app boasts a modern and user-friendly interface that adapts seamlessly to any device, from mobile phones to desktops.

Key Features:

Dynamic Content Sharing: Share content directly to your social media accounts and earn rewards.
Engaging Campaigns: Participate in advertising and social good campaigns directly within the app.
Multilingual Support: Choose between English and French for a more personalized experience.
Responsive Design: Enjoy a smooth experience on any device, be it a mobile phone, tablet, or desktop.
Trusted by Major Brands: Collaborate with well-respected brands and foundations.
Interactive Elements: Enhance your experience with Swiper.js for dynamic slideshows, Accordion menus for FAQs, and more.

Project Structure:

The project is well-organized with a clear structure:

public/: Static files for your application.
src/: Contains the application source code.
assets/: Stores images and icons used throughout the app.
components/: Reusable UI components for building the app's interface.
contexts/: Houses React Context for managing global application state.
pages/: Contains React components representing the application's pages.
styles/: Global and component-specific styles for the app's UI.
utils/: Utility functions and context providers for the application.
index.js: The main entry point of the React application.
.eslintrc.js: ESLint configuration for code style enforcement.
tailwind.config.js: Tailwind CSS configuration file.
vite.config.js: Vite configuration file for building and optimizing the app.
package.json: Manages Node.js dependencies and scripts for the project.
README.md: The file you're currently reading!
